# 📰 Project 5 - AI News Agent

## Overview

The AI News Agent is an automated Python application that collects the latest news headlines from online news sources and prepares a daily news summary.

The goal of this project is to demonstrate how AI-assisted automation can gather information, process it, and deliver useful updates to users through messaging platforms such as WhatsApp.

---

## Objective

To automatically:

- Fetch the latest news headlines
- Select the top 5 important stories
- Format the information into a readable summary
- Deliver updates through WhatsApp every morning

---
