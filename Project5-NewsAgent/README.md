# News Agent

## Objective

Automatically collect the latest news headlines and send the top 5 headlines to WhatsApp every morning.

## Technologies Used

- Python
- NewsAPI
- WhatsApp Automation
- Requests Library

## Workflow

1. Fetch latest news
2. Extract top 5 headlines
3. Format message
4. Send to WhatsApp

## Future Improvements

- AI-based news summarization
- Personalized categories
- Daily scheduling
